# Welcome

My name is Gabriel Luz, I’m 17 years old, and I’m focused on becoming a backend developer.

I’m currently a high school student, studying and building a solid foundation in software development, with an emphasis on backend systems that are stable, scalable, and well-structured.

My main area of study today is **JavaScript with Node.js**, where I focus on backend fundamentals such as:

- HTTP and RESTful APIs  
- Application architecture  
- Data modeling  
- Clean code and maintainability  

I have a strong interest in **software architecture** and system design, especially topics like decoupling, modularity, and long-term maintainability.

In parallel, I study **C and C++** to better understand what happens closer to the machine, including:

- Data structures and algorithms  
- Memory management and pointers  
- How computers work at a low level  
- Performance and optimization concepts  

I’m always working on practical projects to turn theory into real understanding.
